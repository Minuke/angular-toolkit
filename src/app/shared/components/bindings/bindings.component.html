Ejemplos de Binding en Angular

Property Binding

Este ejemplo ilustra Property Binding (de TypeScript a HTML). Cuando el valor de title en el componente cambia, el encabezado se actualiza automáticamente.

Event Binding

Este botón ilustra Event Binding (de HTML a TypeScript). Al hacer clic, se ejecuta la función mostrarMensaje() en el componente.

Two-Way Binding

Hola, {{ user }}!

Este campo de entrada ilustra Two-Way Binding (en ambas direcciones). Cambiar el valor aquí actualiza la propiedad user en el componente, y viceversa: cambios en user actualizan el campo de entrada.
