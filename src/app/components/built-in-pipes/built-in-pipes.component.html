Ejemplos de Pipes Integrados en Angular

Pipes de Transformación de Texto

LowerCasePipe:

Antes: {{ textoMayus }}

→

Después: {{ textoMayus | lowercase }}

Convierte el texto en minúsculas.

UpperCasePipe:

Antes: {{ textoMin }}

→

Después: {{ textoMin | uppercase }}

Convierte el texto en mayúsculas.

TitleCasePipe:

Antes: {{ texto }}

→

Después: {{ texto | titlecase }}

Convierte las palabras del texto en mayúsculas iniciales.

Pipes de Formateo de Números y Fechas

CurrencyPipe:

Antes: {{ number }}

→

Después: {{ number | currency:'USD' }}

Formatea un número como una moneda.

DatePipe:

Antes: {{ today }}

→

Después: {{ today | date:'shortDate' }}

Formatea una fecha en un formato corto.

DecimalPipe:

Antes: {{ number }}

→

Después: {{ number | number:'1.2-2' }}

Formatea el número con decimales controlados.

PercentPipe:

Antes: {{ porcentaje }}

→

Después: {{ porcentaje | percent }}

Convierte un número en un porcentaje.

Pipes de Transformación de Objetos

JsonPipe:

Antes: {{ obj }}

→

Después: {{ obj | json }}

Convierte un objeto en JSON.

KeyValuePipe:

Antes: {{ obj }}

→
  • {{ item.key }}: {{ item.value }}

Convierte un objeto en pares clave-valor.

Otros Pipes

AsyncPipe (Promesa):

Antes: {{ promise }}

→

Después: {{ promise | async }}

Resuelve automáticamente una promesa.

I18nPluralPipe:

Antes: {{ messages.length }}

→

Después: {{ messages.length | i18nPlural: mapping }}

Muestra mensajes pluralizados.

I18nSelectPipe:

Antes: {{ gender }}

→

Después: {{ gender | i18nSelect: genderMap }}

Muestra un texto personalizado basado en el género.

SlicePipe:

Antes: 'abcdef'

→

Después: {{ 'abcdef' | slice:1:4 }}

Extrae una porción de un string.
